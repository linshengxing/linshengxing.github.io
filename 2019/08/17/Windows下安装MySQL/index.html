Windows下安装MySQL

  1. 在MySQL官网下载Community版本的zip压缩文件

  2. 解压，在根目录下新建my.ini配置文件，配置内容如下

  3. 通过命令行工具cmd（管理员）进MySQL的bin目录，输入mysqld -install安装MySQL

  4. 输入mysqld --initialize --user=root --console初始化MySQL数据库

    最后输出的:qdP9gl*jjwa是默认生成的密码

  5. 初始化后，在根目录下多了data目录（用于存放数据）

  6. 启动MySQL服务

  7. 使用默认生成的密码，进入MySQL

  8. 进入MySQL修改默认密码

Windows下的MySQL环境搭建完成！
